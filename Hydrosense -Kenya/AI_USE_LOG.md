# AI_USE_LOG.md — HydroSense-Kenya

This log records every instance where an AI coding assistant was used to support development of
HydroSense-Kenya, in line with the course's Responsible AI Use Requirement. AI output was never
accepted without inspection, testing, or validation against an independent check (analytic result,
NumPy/SciPy verification, or manual trace).

|---|---|---|---|---|---|

## Validation Philosophy

For every AI-supported function or explanation in this project:
1. The output was read and understood line-by-line before acceptance — no AI code was merged unread.
2. Numerical functions were checked against at least one independent reference: a known analytic
   result (e.g. root of `x^2-4`, integral of `x^2`), a NumPy/SciPy built-in (`numpy.linalg.solve`),
   or a manually traced small example.
3. All numerical-method functions in `src/numerical_methods.py` and `src/simulation.py` are covered
   by automated tests in `tests/`, which were run after every modification.
4. Explanatory text generated by AI was rewritten in the team's own words and cross-checked against
   course lecture material before inclusion in any notebook or report.
